# Proactive Threat Forecaster
This is my cybersecurity project on forecasting whether or not a system would be infected by malware or not.

## ABSTRACT
Cybersecurity threats have become more complex and common with the blistering expansion of digital infrastructure and networked systems. The old security systems are mostly reactive in nature and have a tendency of not notifying, or detecting sophisticated or never seen attacks as they occur. 
<br><br>
To address these shortcomings, this project introduces a Proactive Threat Forecaster, which uses the methods of machine learning to detect and forecast possible cyber threats on a proactive basis. The system suggested works on the detection of anomalies in system and network performance as well as the categorization of malicious actions with great precision by analyzing the patterns of their behavior. The preprocessing of the data and extraction of the features are used to ameliorate the performance of the model and alleviate the noise. The historical threat data is used in training a supervised machine learning model that is used to differentiate between normal and malicious behavior. 
<br><br>
System architecture facilitates real time threat monitoring and prediction so that early warning and timely mitigation may take place. Through experiments, it has been proven that the proposed framework can effectively enhance threat detection and minimize false positives. The Proactive Threat Forecaster is an effective and scalable platform that can help strengthen cybersecurity protection in current computing environments
